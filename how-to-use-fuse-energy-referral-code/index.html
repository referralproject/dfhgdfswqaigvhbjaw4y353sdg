How to use a Fuse Energy referral code

If you are signing up to Fuse Energy and want to use the referral code on this site, enter RYAN66827 during the sign-up flow.

Step-by-step

  1. Go to the official Fuse Energy sign-up page or app.
  2. Start your registration and follow the address and tariff steps.
  3. Look for the referral code field in the sign-up flow.
  4. Enter RYAN66827.
  5. Complete registration and review the latest terms before switching.

If you cannot see the referral code field

Sign-up flows sometimes change. If you cannot see the referral field, try restarting the process, checking whether another offer has already been applied, or using the latest version of the app or website.

What the referral offer means

The offer is best described as a chance to win up to £150 bill credit if you meet the current eligibility rules. It is not a guaranteed payment.
